| GPIO | Function | Connected To | Wire Color (suggested) |
|------|----------|--------------|------------------------|
| GPIO4 | DHT22 Data | DHT22 DATA pin | Yellow |
| GPIO16 | HX711 DOUT | HX711 DT pin | Green |
| GPIO17 | HX711 SCK | HX711 SCK pin | Blue |
| GPIO34 | Battery ADC | Voltage divider middle | Orange |
| GPIO2 | Status LED | LED + resistor (optional) | - |
| **GPIO21** | **I2C SDA** | **LCD SDA pin** | **White** |
| **GPIO22** | **I2C SCL** | **LCD SCL pin** | **Purple** |
| **GPIO13** | **Button** | **Push button (to GND)** | **Gray** |
| 3V3 | Power 3.3V | DHT22 VCC, HX711 VCC, LCD VCC | Red |
| GND | Ground | All GND connections | Black |
| VIN | Power Input | TP4056 OUT+ | Red |
